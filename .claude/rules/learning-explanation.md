# 学習用解説ルール

本プロジェクトはAgentCoreワークショップの学びを定着させるための学習用サンプルである。
すべての作業（仕様書・設計書・実装・コードレビュー等）で以下の解説を含めること。

## 必須の解説項目

1. **このステップで学ぶこと** -- 冒頭に記載する
2. **ワークショップ対応** -- AgentCoreワークショップのどのLab（Lab 1〜9）に対応するかを明示する
3. **なぜこう実装するのか** -- 設計判断の理由を説明する
4. **本番構成との違い** -- 簡略化した箇所があればその理由を説明する（参照: `docs/designs/architecture-comparison.md`）

## 適用例

### 仕様書・設計書

各ステップの冒頭に「学習ポイント」セクションを設ける。

### コード実装

- docstring に設計判断の背景を記載する
- 重要な処理にはコメントで「ワークショップ Lab X で学んだ〇〇に対応」と補足する
- README や CLAUDE.md の更新時にも解説を含める
